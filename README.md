# 30 Days of CSS

This plan is designed to help me master CSS in 30 days, with 2-3 hours of daily study and practice. Each day focuses on a specific topic, building on the previous day, with practical exercises to reinforce learning.

## Week 1: Foundations and Basic Styling

### Day 1: CSS Basics and Syntax

- **Learn**: What is CSS? Syntax (selectors, properties, values). Including CSS (inline, internal, external).
- **Practice**: Style a simple HTML page with `color`, `background-color`, and `font-size`.
- **Resource**: MDN Web Docs (CSS Basics).

### Day 2: Selectors (Part 1)

- **Learn**: Element, class, and ID selectors. Basic combinators (descendant, child).
- **Practice**: Create a page with multiple elements styled using class and ID selectors.
- **Resource**: CSS-Tricks (Selectors Guide).

### Day 3: Box Model

- **Learn**: Content, padding, border, margin. `box-sizing: content-box` vs. `border-box`.
- **Practice**: Build a card component with padding, margins, and borders.
- **Resource**: MDN (CSS Box Model).

### Day 4: Text and Typography

- **Learn**: `font-family`, `font-size`, `font-weight`, `text-align`, `line-height`.
- **Practice**: Style a blog post with headings and paragraphs using typography properties.
- **Resource**: W3Schools (CSS Fonts).

### Day 5: Colors and Backgrounds

- **Learn**: Color formats (hex, RGB, RGBA). `background-color`, `background-image`, `background-size`.
- **Practice**: Create a hero section with a solid color and a background image.
- **Resource**: MDN (CSS Colors).

### Day 6: Display and Positioning

- **Learn**: `display: block`, `inline`, `inline-block`. Positioning: `static`, `relative`, `absolute`.
- **Practice**: Build a fixed header and a positioned button.
- **Resource**: CSS-Tricks (Positioning).

### Day 7: Mini-Project: Simple Webpage

- **Task**: Combine Days 1-6 to create a styled webpage (e.g., a personal bio page) with text, colors, and basic layout.
- **Practice**: Use DevTools to debug styles.
- **Resource**: freeCodeCamp (CSS Challenges).

## Week 2: Layouts and Selectors

### Day 8: Selectors (Part 2)

- **Learn**: Pseudo-classes (`:hover`, `:focus`). Attribute selectors.
- **Practice**: Style a navigation menu with hover effects.
- **Resource**: MDN (Pseudo-classes).

### Day 9: Specificity and Combinators

- **Learn**: CSS specificity rules. Advanced combinators (`+`, `~`).
- **Practice**: Create a layout with nested elements styled using combinators.
- **Resource**: CSS-Tricks (Specificity).

### Day 10: Flexbox (Part 1)

- **Learn**: `display: flex`, `flex-direction`, `justify-content`, `align-items`.
- **Practice**: Build a centered content section using Flexbox.
- **Resource**: CSS-Tricks (Flexbox Guide).

### Day 11: Flexbox (Part 2)

- **Learn**: `flex-wrap`, `flex-grow`, `flex-shrink`, `flex-basis`.
- **Practice**: Create a responsive card layout with Flexbox.
- **Resource**: Flexbox Froggy (Game).

### Day 12: CSS Grid (Part 1)

- **Learn**: `display: grid`, `grid-template-columns`, `grid-template-rows`, `gap`.
- **Practice**: Build a simple 2x2 grid layout for a photo gallery.
- **Resource**: CSS-Tricks (Grid Guide).

### Day 13: CSS Grid (Part 2)

- **Learn**: Named grid areas, `grid-template-areas`, `minmax()`, `auto-fit`.
- **Practice**: Create a dashboard layout with header, sidebar, and content.
- **Resource**: Grid Garden (Game).

### Day 14: Mini-Project: Portfolio Layout

- **Task**: Build a portfolio page using Flexbox and Grid, with a header, project cards, and footer.
- **Practice**: Ensure consistent spacing and alignment.
- **Resource**: Frontend Mentor (Portfolio Challenge).

## Week 3: Responsive Design and Interactivity

### Day 15: Responsive Design Basics

- **Learn**: Relative units (`%`, `vw`, `vh`, `rem`, `em`). Viewport meta tag.
- **Practice**: Convert a fixed-width layout to use relative units.
- **Resource**: MDN (Responsive Design).

### Day 16: Media Queries

- **Learn**: Media query syntax, breakpoints, mobile-first approach.
- **Practice**: Make a webpage responsive with at least 3 breakpoints.
- **Resource**: W3Schools (Media Queries).

### Day 17: Responsive Images

- **Learn**: `srcset`, `picture` element, `max-width` for images.
- **Practice**: Add responsive images to a gallery layout.
- **Resource**: MDN (Responsive Images).

### Day 18: Transitions

- **Learn**: `transition-property`, `transition-duration`, `transition-timing-function`.
- **Practice**: Create buttons with hover transitions (e.g., color, scale).
- **Resource**: MDN (CSS Transitions).

### Day 19: Animations

- **Learn**: `@keyframes`, `animation-name`, `animation-duration`, `animation-iteration-count`.
- **Practice**: Build a loading spinner or animated text effect.
- **Resource**: CSS-Tricks (Animations).

### Day 20: Transform Properties

- **Learn**: `translate`, `rotate`, `scale`. Combining with transitions.
- **Practice**: Create a card flip effect on hover.
- **Resource**: MDN (Transforms).

### Day 21: Mini-Project: Interactive Landing Page

- **Task**: Build a responsive landing page with hover animations, Flexbox/Grid, and media queries.
- **Practice**: Test on multiple devices using DevTools.
- **Resource**: Scrimba (CSS Projects).

## Week 4: Advanced Techniques and Real-World Application

### Day 22: CSS Variables

- **Learn**: Custom properties (`--variable`), using with `var()`.
- **Practice**: Create a theme switcher (e.g., light/dark mode).
- **Resource**: MDN (CSS Variables).

### Day 23: Pseudo-Elements

- **Learn**: `::before`, `::after` for decorative content.
- **Practice**: Add decorative elements to a button or card (e.g., icons, shapes).
- **Resource**: CSS-Tricks (Pseudo-Elements).

### Day 24: Accessibility in CSS

- **Learn**: Focus styles, ARIA roles, contrast ratios, semantic HTML with CSS.
- **Practice**: Style a form with accessible focus states and high contrast.
- **Resource**: WebAIM (Accessibility Basics).

### Day 25: CSS Architecture

- **Learn**: BEM methodology, organizing CSS files, avoiding `!important`.
- **Practice**: Refactor a small project using BEM naming.
- **Resource**: BEM Documentation.

### Day 26: Introduction to CSS Frameworks

- **Learn**: Tailwind CSS basics (utility classes, responsive design).
- **Practice**: Build a simple layout using Tailwind CSS.
- **Resource**: Tailwind CSS Docs.

### Day 27: Introduction to Sass

- **Learn**: Sass basics (variables, nesting, mixins).
- **Practice**: Convert a CSS file to SCSS with variables and nesting.
- **Resource**: Sass Basics (sass-lang.com).

### Day 28: Debugging and Optimization

- **Learn**: Using DevTools to inspect and debug CSS. Minimizing reflows, optimizing selectors.
- **Practice**: Debug a broken layout and optimize CSS file size.
- **Resource**: MDN (DevTools Guide).

### Day 29: Final Project (Part 1)

- **Task**: Start a complete website (e.g., e-commerce product page or portfolio) using all learned concepts.
- **Practice**: Plan the layout, apply Flexbox/Grid, ensure responsiveness.
- **Resource**: Frontend Mentor (Project Ideas).

### Day 30: Final Project (Part 2)

- **Task**: Complete and polish the website. Add animations, accessibility, and test cross-browser compatibility.
- **Practice**: Deploy to a free host (e.g., GitHub Pages) and share on X for feedback.
- **Resource**: GitHub Pages (Deployment Guide).

- **Resources**:
